# Official vs Community Urban Mapper

UrbanMapper Community extends the official Urban Mapper with select enhancements. Use these tabs to see how each pipeline
stage compares today. We keep this page up to date as the Community adds new capabilities.

=== "Loader"

    **Community additions**

    - None yet. The Community build follows the official Urban Mapper loaders.

    **References**

    - Not applicable.

=== "Urban Layer"

    **Community additions**

    - Uber H3 hexagon tessellation layer for grid-based analyses.

    **References**

    - [UberH3Grid API reference](api/urban_layers.md#urbanmappermodulesurban_layeruberh3grid).

=== "Imputer"

    **Community additions**

    - None yet. The Community build follows the official Urban Mapper imputers.

    **References**

    - Not applicable.

=== "Filter"

    **Community additions**

    - None yet. The Community build follows the official Urban Mapper filters.

    **References**

    - Not applicable.

=== "Enricher"

    **Community additions**

    - None yet. The Community build follows the official Urban Mapper enrichers.

    **References**

    - Not applicable.

=== "Visualiser"

    **Community additions**

    - Lonboard-based visualisers for high-performance web maps via GPU.

    **References**

    - [Lonboard visualiser APIs](api/visualisers.md#urbanmappermodulesvisualiserlonboardclassicvisualiser).
